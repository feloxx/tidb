按照以往的经验,一个程序的入口都是先找main函数;java也好,go也同样如此,我们先找到main在哪.

```go
func main() {
  flag.Parse()
  if *version {
    printer.PrintRawTiDBInfo()
    os.Exit(0)
  }

  registerStores()
  loadConfig()
  overrideConfig()
  validateConfig()
  setGlobalVars()
  setupLog()
  setupTracing() // Should before createServer and after setup config.
  printInfo()
  setupBinlogClient()
  setupMetrics()
  createStoreAndDomain()
  createServer()
  setupSignalHandler()
  runServer()
  cleanup()
  os.Exit(0)
}
```
